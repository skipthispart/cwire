# CWire

A streamlined web application designed to provide users with reliable, up-to-date VPN configurations across multiple modern protocols. The platform is built around simplicity and speed — the moment you land on the site, you have immediate access to a curated set of subscription links and individual configs, all ready to be copied and integrated into your preferred client.

## What it does

- Aggregates free VPN configurations from publicly available sources
- Continuously refreshes its config pool so users always have working endpoints
- Supports all major modern protocols: VLESS, VMess, Trojan, Shadowsocks, Hysteria2, and TUIC
- Provides direct subscription links that can be imported into any compatible client
- Delivers a clean, distraction-free interface with zero friction between the user and the configs they need

## Design philosophy

The project was not built to be another bloated VPN dashboard. The goal was straightforward — create a minimal, fast, and reliable bridge between users and the configurations they rely on daily. Every decision, from the layout to the caching strategy, was made with one principle in mind: get the user to a working config in the shortest possible time, without unnecessary steps, signups, or interruptions.

## Privacy

No user accounts, no tracking, no analytics, no cookies. The application does not collect, store, or transmit any personal information. Visitor counts are aggregated anonymously, and all configuration data is fetched on-demand from upstream sources. What you do on the site stays on the site.
